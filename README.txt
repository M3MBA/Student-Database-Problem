Double Hashing
    T is the input size
    insert O(T)
    update O(T)
    delete O(T)
    contains O(T)
    get O(T)
    address O(T)


Single Chaining with BST
    h is height of BST
    insert 
        finding index O(1) inserting element in BST in O(h)
    update
        finding index O(1), updating element in O(h)
    delete
        finding index O(1), deleting element in O(h)
    contains 
        O(h)
    get
        O(h)
    address
        O(h)
